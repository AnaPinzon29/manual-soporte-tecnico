Manual Tecnico de Soporte de Reparación

En este manual podrás observar y aprender, algunas cosas sobre el mantenimiento de dispositivos moviles.

CELULAR

Paso 1

Comenzamos quitandole la tapa (esto depende mucho de el modelo de celular)... Para esto necesitaremos destornilladores y pinzas para poder coger mejor los componentes que trae y no dañarlos...

Paso 2

Levantamos con las pinzas y mucha precaución los cables también conocidos como buses, retiramos la pila

Paso 3

Ponemos el celular con la pantalla hacia abajo para empezar a darle calor con la plancha separadora -Recomendación: Utilizar guantes para evitar quemarse en el proceso.

Paso 4

Ya caliente el display, empezamos a despegar el display con las palancas de apertura, las cuales se pueden observar en la imagen.

Paso 5

Retiramos con precaución el display y empezamos a desarmarlo

Para finalizar

Aquí podemos observar los componentes y algunas partes que están dentro del dispotivo.

HERRAMIENTAS UTILIZADAS

Estás son las herramientas que se utilizaron en el proceso

Destornilladores: Son herramientas con una punta en forma de cruz o de cabeza plana que se utilizan para quitar los tornillos que sujetan las partes del teléfono celular, incluido el display.

Palanca de apertura: Es una herramienta plana y delgada, generalmente hecha de plástico o metal, que se utiliza para separar las carcasas o componentes pegados del teléfono sin dañarlos. Ayuda a abrir el dispositivo sin rayar o romper los bordes.

Pinzas: Las pinzas de precisión son herramientas pequeñas y finas con puntas afiladas y agarradoras. Se utilizan para manipular y sujetar componentes pequeños, como cables o tornillos, durante el proceso de desmontaje.

Cuchillas de precisión: Estas cuchillas tienen una hoja delgada y afilada que se utiliza con cuidado para cortar adhesivos, retirar pegatinas o levantar componentes. Se deben utilizar con precaución para evitar lesiones o daños al dispositivo.

Ventosa: Una ventosa es una herramienta que se utiliza para levantar el display del teléfono celular. Se coloca sobre la pantalla y se tira de ella mediante la generación de vacío para separarla de la carcasa y los adhesivos.
